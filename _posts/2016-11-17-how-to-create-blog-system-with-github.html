假设你已经有了一个github账号，可以正常创建项目，且在本机上已经安装了git

创建一个blog项目，比如我的叫myblog

使用命令行工具创建一个git项目
mkdir blog_github
cd blog_github/
git init

创建一个分支
git checkout --orphan gh-pages

创建一个config文件“_config.yml”
vim _config.yml
输入以下内容
baseurl: /myblog
其中myblog要与项目名一致

创建layout文件夹“_layouts”
mkdir _layouts
cd _layouts/

新建一个html vim default.html
输入以下内容
<!DOCTYPE html>


　
　　　　{{ page.title }}


　　　　{{ content }}


去上一级目录 cd .. 创建文件夹_posts mkdir _posts 进入_posts cd _posts 创建文件2016-11-17-hellow-world-example.html(格式必须是yyyy-mm-dd-filename.html) vim 2016-11-17-hellow-world-example.html 输入以下内容 --- layout: default title: 你好，世界 ---

{{ page.title }}
